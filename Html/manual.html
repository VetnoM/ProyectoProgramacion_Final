Manual de Usuario

🏨 Sistema de Gestión Hotelera

👶 Guía rápida

  1. 🏠 Pantalla de bienvenida:
    • Al abrir la aplicación, verás un menú con botones para cada módulo: Personas, Habitaciones, Reservas, Tareas y Facturación.
    • Haz clic en el módulo que quieras gestionar.
  2. 👤 Agregar personas (clientes y empleados):
    • Selecciona si quieres registrar un Cliente, Empleado o Ambos.
    • Rellena los datos obligatorios (nombre, apellido, documento, etc.).
    • Pulsa Agregar para guardar.
    • Para modificar, busca por documento, selecciona y edita los datos, luego pulsa Guardar Cambios.
  3. 🛏️ Gestionar habitaciones:
    • Rellena los datos de la habitación y pulsa Crear.
    • Para modificar o eliminar, selecciona una habitación de la lista y usa los botones correspondientes.
  4. 📅 Crear y facturar reservas:
    • Selecciona habitación, cliente, tipo de reserva y fechas.
    • El sistema calcula el precio automáticamente.
    • Pulsa Crear Reserva.
    • Para facturar:
      • Selecciona la reserva y pulsa Facturar.
      • Elige el método de pago y confirma.
  5. 📝 Gestionar tareas:
    • Crea tareas con descripción y fechas.
    • Asigna tareas a empleados y marca como completadas cuando terminen.
  6. 💡 Consejos útiles:
    • Los campos obligatorios están marcados o son requeridos para guardar.
    • Si tienes dudas, pasa el ratón sobre los botones para ver una breve descripción.
    • El sistema te avisará si intentas hacer algo incorrecto (por ejemplo, reservar una habitación ya ocupada).
  7. ❗ ¿Ves un error?
    • Lee el mensaje en pantalla y revisa los campos.
    • Si el problema persiste, consulta la sección de incidencias o pide ayuda a soporte.

❓ ¿Qué es este programa?

Sistema de Gestión Hotelera es una aplicación de escritorio moderna y fácil de usar, desarrollada en JavaFX, que permite gestionar clientes, empleados, habitaciones, reservas, tareas y facturación de manera centralizada y eficiente.

👤 Personas
  • Alta y modificación de clientes y empleados
  • Validación y búsqueda rápida
🛏️ Habitaciones
  • Creación y edición
  • Control de estado y precios
📅 Reservas
  • Evita solapamientos
  • Cálculo automático de precios
📝 Tareas
  • Asignación y seguimiento
  • Filtros y búsqueda avanzada
💳 Facturación
  • IVA y descuentos automáticos
  • Actualización de estado

🚦 ¿Cómo funciona? - Pasos principales

  1. 🏠 Inicio y navegación:
    • Desde el menú principal accede a cada módulo.
    • La navegación es lateral e intuitiva.
  2. 👤 Gestión de Personas:
    • Registra clientes, empleados o ambos.
    • Validación automática de datos.
    • Modifica y busca usuarios fácilmente.
  3. 🛏️ Gestión de Habitaciones:
    • Crea habitaciones con tipo, capacidad y precios.
    • El estado se actualiza según reservas y limpieza.
  4. 📅 Gestión de Reservas:
    • Selecciona habitación, cliente y fechas.
    • Evita solapamientos y calcula el precio.
    • Facturación integrada.
  5. 📝 Gestión de Tareas:
    • Crea y asigna tareas a empleados.
    • Marca tareas como completadas.
  6. 💳 Facturación:
    • Genera facturas con IVA y descuentos.
    • Actualiza el estado de la reserva automáticamente.

❔ Preguntas frecuentes

  • ¿Puedo registrar a una persona como cliente y empleado a la vez?
    Sí, seleccionando la opción Ambos al registrar la persona.
  • ¿Cómo se calcula el precio de una reserva?
    Automáticamente según el tipo de reserva (AD o MP), la duración y el tipo de cliente.
  • ¿Qué sucede si intento reservar una habitación ya ocupada?
    El sistema no permite solapamientos y te avisará si la habitación ya está reservada en esas fechas.
  • ¿Puedo aplicar descuentos en la facturación?
    Sí, si el método de pago es efectivo, se aplica un descuento automático del 10%.
  • ¿Cómo se gestionan las tareas?
    Puedes crear tareas, asignarlas a empleados y marcar su estado. El sistema actualiza automáticamente el estado global cuando todos los empleados la completan.
